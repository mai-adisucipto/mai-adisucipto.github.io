===============================================
PANDUAN PENEMPATAN GAMBAR
Masjid Al Ikhlas Adi Sucipto – Landing Page
===============================================

Letakkan file gambar Anda di folder ini dengan nama persis seperti berikut:

1. FOTO MASJID (Hero Section)
   Nama file : masjid.jpg  (atau .png / .webp)
   Ukuran    : Minimal 800x600px, rasio 4:3 dianjurkan
   Keterangan: Foto utama masjid yang tampil di hero section kanan

2. FOTO KEGIATAN (Gallery Section – 4 kotak)
   Nama file : kegiatan-1.jpg  → Label: Kajian
   Nama file : kegiatan-2.jpg  → Label: Daurah / Muslimah
   Nama file : kegiatan-3.jpg  → Label: Kegiatan
   Nama file : kegiatan-4.jpg  → Label: Kajian Rutin
   Ukuran    : Minimal 600x450px, rasio 4:3 dianjurkan

3. QRIS (Donasi Section)
   Nama file : qris.png  (atau .jpg)
   Ukuran    : Minimal 400x400px, rasio 1:1 (kotak)
   Keterangan: Gambar QRIS resmi masjid untuk di-scan jamaah

===============================================
Format yang didukung: .jpg, .jpeg, .png, .webp
